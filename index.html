Read The Surface In Layers

Stream does not ask for belief first. It starts with measurable parts: distribution, parity, inequality, network structure, and long-range trend. The aim is not to flatten culture into a number. The aim is to make pressure visible enough to inspect.

The tabs move from overview to rows, from findings to field knowledge, from methods to verdict, and from modeled representation to a separate real-data music lane. Each route exists so the reader can choose a stronger path instead of being trapped in one story.

How To Use It

  • Move the year: watch one system change instead of comparing disconnected screenshots of progress.
  • Filter the field: ask whether a pattern is broad or tied to a platform, genre, or production culture.
  • Use methods with findings: the field knowledge and the math are meant to correct each other.
  • Switch to music virality: use the real-data lane when you want the public attention layer first.

Technical Implementation — What Actually Runs

Metrics are computed live by the Python backend: Shannon entropy for diversity, a distance-from-parity index for gender balance, chi-square statistics for dialogue / age / racial bias, intersectional representation ratios against a uniform baseline, and assortativity coefficients on character interaction graphs (NetworkX). Before a claim lands, the app also checks whether the same signal survives when you slice the data by year, platform, genre, and media type. Visualizations use a real D3.js force simulation (charge, spring, and collision forces), Chart.js canvas rendering, and Plotly.js including WebGL-accelerated 3D. The demo dataset is synthetic, generated by a seeded, documented statistical model — see the Learn tab for the honesty note.

How big is each bias? (effect size)

A chi-square test tells you a dialogue skew is unlikely to be chance, but its value balloons with sample size. Cramér’s V rescales it to 0–1 so you can compare magnitudes and read a plain-language band (negligible → large).

The decade trend, fitted

Instead of comparing only two years, this fits an ordinary least-squares line across every year in the dataset. The slope is the average change per year; R² says how much of the movement the straight-line trend explains.

The Power Law of Attention

View counts are not normally distributed — they follow a power law where a tiny elite captures most of the attention. The complementary cumulative distribution (CCDF) on log-log axes reveals the heavy tail: a straight line means pure power-law behaviour. We care because discovery systems, ad revenue, and cultural memory all behave differently when attention is this concentrated.

Correlation Architecture

What actually correlates with more views? Spearman rank correlations show raw associations; partial correlations strip out the effect of channel size so you can see what matters independently of how big the channel already is. That helps separate reach from structure, which is the difference between “popular because it is everywhere” and “popular because something about it travels.”

About This Published Surface

Stream is a public Loopchii research surface for measuring representation asymmetries, attention concentration, and the social shape of media systems. The representation dashboard runs on a clearly labeled synthetic dataset so the math stays reproducible. The Music Virality module is separate because it is built on real YouTube data and should be read differently.
