Technical Implementation — What Actually Runs

Metrics are computed live by the Python backend: Shannon entropy for diversity, a distance-from-parity index for gender balance, chi-square statistics for dialogue / age / racial bias, intersectional representation ratios against a uniform baseline, and assortativity coefficients on character interaction graphs (NetworkX). Visualizations use a real D3.js force simulation (charge, spring, and collision forces), Chart.js canvas rendering, and Plotly.js including WebGL-accelerated 3D. The demo dataset is synthetic, generated by a seeded, documented statistical model — see the Learn tab for the honesty note.

How big is each bias? (effect size)

A chi-square test tells you a dialogue skew is unlikely to be chance, but its value balloons with sample size. Cramér’s V rescales it to 0–1 so you can compare magnitudes and read a plain-language band (negligible → large).

The decade trend, fitted

Instead of comparing only two years, this fits an ordinary least-squares line across every year in the dataset. The slope is the average change per year; R² says how much of the movement the straight-line trend explains.

About This Analysis

StreamLens Analytics quantifies representation asymmetries in streaming media using statistical and network-analysis methods, computed live on a clearly-labeled synthetic dataset. The Music Virality module analyses 100 real YouTube music videos (10.59 billion views) from an actual 2025 dataset.
