# Code of Conduct

Stream is a public research repository. People should be able to inspect it, question it, and contribute to it without being talked down to.

## Our standard

We want this project to feel:

- rigorous
- generous
- calm under disagreement
- readable for newcomers
- serious enough for experienced engineers and researchers

## Expected behavior

Contributors are expected to:

- critique ideas without attacking people
- keep claims tied to evidence
- respect the project's public and private boundaries
- accept correction when a statement, source line, or method is wrong
- make room for contributors who are still learning

## Unacceptable behavior

The following do not belong here:

- harassment or intimidation
- personal attacks
- discriminatory language
- bad-faith review behavior
- doxxing or exposure of private information
- repeated attempts to push private LOOPCHii architecture into the public repo
- knowingly overstating what the code or data proves

## Scope

This code of conduct applies to:

- repository issues
- pull requests
- code review comments
- discussions connected to this project

## Reporting

If something crosses the line, open a private or public issue depending on what is safest and most appropriate.

When reporting, include:

- what happened
- where it happened
- what response would help resolve it

## Stewardship

Project maintainers may remove comments, close contributions, or restrict participation when behavior makes the repository less safe or less trustworthy.

The goal is not punishment theater. The goal is to keep the project usable, honest, and constructive.
